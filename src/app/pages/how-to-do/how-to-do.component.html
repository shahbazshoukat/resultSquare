How To Find Results?

Latest Results

Navigate to resultsquare.pk and find latest results updates.

Results can be filtered by province, class/grade and board/university.

Quick Results Filter

Find any result from Quick Filter panel.

Select your class/grade, university, year, exam type and navigate to result page instantly.

Result Information

Basic information related to a result can be found in Result Information section.

Result Page

On a result page you can find relevant board/university website page.
